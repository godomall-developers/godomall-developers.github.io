### 기능 추가
- 파일 업로드 보안 강화 (#15013)
  - 파일 업로드 시 보안 검증 로직이 강화됐어요.
  - 업로드 파일의 안전성이 향상돼 더욱 안전하게 파일을 관리할 수 있어요.

### 버그 수정
- 게시판 및 상품 안내 보안 강화 (#15024)
  - 게시판 관리와 상품 상세 안내 기능의 보안이 더욱 강화됐어요.
  - 안전한 서비스 운영을 위해 보안 취약점이 개선됐어요.
- 플러스리뷰 위젯 보안 강화 (#15045)
  - 플러스리뷰 위젯의 미리보기 기능에서 보안이 강화됐어요.
  - 더욱 안전한 서비스 이용이 가능해졌어요.
- 구매 리뷰 보안 강화 (#15102)
  - 구매 리뷰 등록 시 주문 상품의 소유권을 검증하는 보안이 강화됐어요.
  - 더 안전한 리뷰 시스템을 이용할 수 있어요.
- 정렬 조건 보안 강화. (#15193)
  - SQL Injection 방지를 위해 정렬 조건에 화이트리스트 방식이 적용돼요.
  - 허용된 정렬 옵션만 사용 가능하도록 보안이 강화됐어요.
- 상품 검색 보안 강화. (#15404)
  - 상품 검색 페이지의 페이지 번호 처리 방식이 더욱 안전하게 개선됐어요.
- 상품 추천 기능 보안 강화. (#15405)
  - 상품 추천 기능의 데이터 처리 과정이 더욱 안전하게 개선됐어요.
- 뱅크다 보안 강화. (#15410)
  - 뱅크다 입금 내역 조회 시 보안이 강화됐어요.
  - 안전한 결제 환경을 위해 처리 방식이 개선됐어요.

### 기능 변경
- 에러 로그 처리기(ErrorProcessor) 도입 및 LogAndCrash 적재 포맷 개선 (#15640)
  - 에러 발생 시 파일/라인/스택트레이스를 자동으로 기록하는 로그 처리기를 도입해, 오류 발생 원인을 더 빠르게 추적할 수 있어요.
  - 불필요한 로그 데이터를 정리하고 에러 유형 분류 로직을 개선해 로그 적재가 더 안정적으로 처리돼요.
- 고도몰 어드민 메인의 운영필수서비스 및 부가서비스 페이지 개선 (#15748)
  - 고도몰 어드민 메인에서 운영필수서비스와 부가서비스 페이지가 개선됐어요.
  - 더 편리하게 서비스를 확인하고 이용할 수 있도록 랜딩 페이지와 제목이 변경됐어요.
