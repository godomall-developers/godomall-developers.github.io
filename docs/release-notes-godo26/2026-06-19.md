### 기능 추가
- 반응형 스킨에서 마이앱 지원 및 모바일샵 설정 유효성 검사 추가 (#14261)
  - 반응형 스킨에서도 마이앱을 이용할 수 있도록 스크립트가 추가되었어요. 스킨 설정과 모바일샵 사용여부 설정의 불일치를 방지하는 유효성 검사도 함께 추가되었어요.
- 간편상품등록 기능 추가 (#14549)
  - 카테고리·옵션 등 주요 상품 정보를 하나의 페이지에서 빠르게 등록할 수 있는 간편상품등록 기능이 제공되어요. 기존 상품 옵션 불러오기·자주 쓰는 옵션 저장 등 편의 기능으로 상품 등록 시간을 단축할 수 있어요.
- 개발소스관리 스테이징 환경 개선 및 배포 검증 강화 (#15088)
  - 스테이징 상점 만료 기간 표시, 반응형 스킨 자동 마이그레이션, 배포 전 DML 스키마 검증 기능이 추가되었어요. 작업소스 검증 및 원본소스 diff 확인, 사후 검증 인트로 자동 해제 등 배포 프로세스 전반이 개선되었어요.
- 방문자 통계 분석 기반 세션 기반으로 전환 (#15202)
  - 방문자 통계의 집계 및 분석 기반이 세션(CID/SID) 기반으로 전환되어, 더 정확한 방문자 통계를 제공하게 되었어요.
- 관리자 내 마케팅 콘텐츠 노출 영역 신설 (#15221)
  - NHN커머스의 마케팅 콘텐츠를 강조해 노출할 수 있는 전용 영역이 관리자에 추가되었어요.

### 버그 수정
- PC 브라우저로 모바일 도메인 접속 시 이니시스 결제 및 취소 개선 (#13327)
  - PC 브라우저로 모바일 도메인에 접속해 이니시스로 결제하거나 취소할 때 정상적으로 동작하도록 개선되었어요.
- 토스페이 결제 시 재고 부족 시 결제-주문 상태 일관성 보장 (#13533)
  - 토스페이 결제 완료 후 재고가 부족한 경우, 수동 승인 방식이 적용되어 결제가 확정되지 않고 처리되어요. 결제 상태와 주문 상태가 일관되게 관리되어요.
- 상품 리스트, 기획전, 플러스리뷰 페이지 보안 강화 (#14539)
  - 상품 리스트 페이지, 기획전 페이지, 플러스리뷰 리스트 페이지의 보안이 강화되었어요.
- SMS 대체 발송 자동 알림 키 중복 방지 강화 (#15015)
  - SMS 대체 발송 자동 알림 키가 중복으로 생성되지 않도록 처리가 강화되었어요.
- 상품 리스트 페이지 보안 강화 (#15093)
  - 상품 리스트 페이지의 보안이 강화되었어요.
- 반응형 스킨 마이앱 딥링크 정확도 개선 (#15216)
  - 반응형 스킨에서 메인 이외의 페이지를 딥링크로 마이앱에 진입할 때, 의도한 페이지로 정확히 이동하도록 개선되었어요.
- 사은품 지급 수량 최대값 확대 (127 → 10,000개) (#15309)
  - 일반/정기 상품의 사은품 지급 수량 최대값이 10,000개로 통일되어, 더 많은 수량의 사은품을 설정할 수 있어요.
- 주문 카운트 토픽 발행 안정성 강화 (#15322)
  - 주문 카운트 대상이 없을 때는 토픽이 발행되지 않도록 처리가 추가되어, 불필요한 빈 데이터 발행이 방지되었어요.
- 외부 저장소 게시판에서 파일 미첨부 시 글 등록 개선 (#15422)
  - 게시판 저장 위치를 외부 저장소로 설정한 경우에도 파일 첨부 없이 글·답변을 정상적으로 등록할 수 있도록 개선되었어요.
- 바로구매 후 로그인 과정의 구매 진행 안내 개선 (#15423)
  - 상품을 바로구매 후 로그인 시, 정상 판매 중인 상품에서 구매 안내가 의도대로 표시되도록 개선되었어요. 외부망·모바일 기기에서 PC 화면으로 접속하는 환경에서도 정상적으로 구매가 진행되어요.
- 처리 완료된 클레임의 상태값 보호 강화 (#15540)
  - 환불 완료·승인·거절 처리된 클레임에 추가 처리를 시도하면 안내 메시지가 노출되어 의도치 않은 상태 변경을 방지할 수 있어요.
- 결제 취소 시 결제 진행 상태 안정적으로 정리 (#15541)
  - 고객이 결제를 취소할 때 결제 진행 여부 세션이 정상적으로 정리되도록 개선되었어요.
- 취소복원 시 재고 부족으로 복원이 보류되면 마일리지도 함께 보류되도록 개선 (#15544)
  - 주문 취소복원 시 재고가 부족한 경우, 마일리지 복원도 함께 보류되도록 처리가 개선되었어요.
- 재고 복구 처리 안정성 개선 (#15546)
  - 재고 복구 중 쿼리 처리 안정성이 향상되어 정상적으로 복구가 완료되어요.

### 기능 변경
- 주문 다운로드 엑셀에 '배송번호' 항목 추가 및 '배송비코드' 명칭 변경 (#15539)
  - 주문 내역 다운로드 양식 관리에서 배송번호(orderDeliverySno)를 선택할 수 있게 되었고, '배송비코드' 명칭도 변경되었어요.
